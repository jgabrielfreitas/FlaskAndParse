# FlaskAndParse
Integration with Facebook Parse using Flask and Python 2.7

### Install [Flask]
```terminal
$ pip install Flask
$ python hello.py
 * Running on http://localhost:5000/
```

### Run a Select * from Parse
```terminal
$ python hello_flask.py
```
Access in http://127.0.0.1:5000/parse_all
